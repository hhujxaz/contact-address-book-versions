# Address Book Management System
A command-line contact management application built with Python.

## Features
- Add, view, search, edit, and delete contacts
- Support for Personal and Business contact types
- CSV-based data storage
- Search by name or phone number

## Requirements
- Python 3.x
- No external libraries required (uses built-in csv module)

## How to Run
python contact_book2.py

## Usage
Follow the menu prompts to manage your contacts.
